# 📝 待办事项清单应用 (To-Do List Application)

一个功能完整的待办事项清单应用，具有本地存储功能。所有数据自动保存到浏览器的 localStorage。

## 🌟 功能特性

- ✅ **添加待办事项** - 快速添加新的任务
- ✅ **完成标记** - 勾选完成任务
- ✅ **删除任务** - 移除不需要的待办事项
- ✅ **筛选功能** - 查看全部、未完成或已完成的任务
- ✅ **统计信息** - 显示总计、已完成和未完成的数量
- ✅ **本地存储** - 数据自动保存，刷新页面后仍保持
- ✅ **清除选项** - 清除已完成或全部任务
- ✅ **响应式设计** - 适配各种屏幕尺寸
- ✅ **美观界面** - 现代化渐变设计和流畅动画

## 🚀 快速开始

### 在线使用
直接在浏览器中打开 `index.html` 文件即可使用。

### 本地存储说明
- 所有待办事项自动保存到浏览器的 localStorage
- 清除浏览器缓存后数据会被删除
- 支持多标��页，数据在同一浏览器中全局同步

## 📱 使用说明

### 基本操作
1. **添加任务** - 在输入框中输入任务内容，点击"添加"或按 Enter
2. **完成任务** - 勾选任务前的复选框标记为完成
3. **删除任务** - 点击任务右侧的垃圾桶图标删除
4. **筛选任务** - 使用顶部的筛选按钮查看不同状态的任务

### 高级功能
- **清除已完成** - 一键删除所有已完成的任务
- **清除全部** - 删除所有任务（需要二次确认）
- **实时统计** - 自动统计总任务数、已完成数和未完成数

## 💾 LocalStorage 实现

应用使用浏览器的 LocalStorage API 来持久化存储数据：

```javascript
// 保存数据
localStorage.setItem('todos', JSON.stringify(todos));

// 加载数据
const todos = JSON.parse(localStorage.getItem('todos'));
```

## 📝 常见问题

**Q: 数据会丢失吗？**
A: 只要不清除浏览器缓存，数据会一直保存。

**Q: 可以在不同设备间同步吗？**
A: 不可以，本地存储仅限当前设备和浏览器。

**Q: 任务有字数限制吗？**
A: 有，最多 100 个字符。
